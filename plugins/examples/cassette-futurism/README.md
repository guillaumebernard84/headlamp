# Cassette Futurism Theme

A simple retro-futuristic theme inspired by cassette futurism aesthetics.

Features warm beige/tan tones, retro orange accents, and a vintage tech feel.

## Usage

```bash
cd plugins/examples/cassette-futurism
npm install
npm start
```

Then go to Settings > General and select "cassette futurism" from the theme dropdown.
